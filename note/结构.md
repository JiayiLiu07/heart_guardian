# 🗂️ CardioGuard-AI 项目结构（最小增删后）



## 🔺 根目录
| 文件 | 作用 |
| ---- | ---- |
| `app.py` | 主入口：路由守卫 + 全局 CSS 变量（--primary/--accent/--success）注入 |
| `train.py` | 训练 & 重新生成 4 张图表 → assets/ |
| `requirements.txt` | 依赖清单（新增 streamlit-quill、func-timeout） |
| `.env` | 存 API_KEY、DB_URL 等敏感信息 |

---

## 📊 data/
| 文件 | 作用 |
| ---- | ---- |
| `cardio_train.csv` | 原始训练集，供 `data_viz.py` 做四图可视化 |

---

## 📁 assets/（训练产物 + 兜底素材）
| 文件 | 作用 |
| ---- | ---- |
| `xgb_risk.pkl` | 心脏总体风险模型 |
| `xgb_subtype.pkl` | 15 类疾病子模型 |
| `recipe_pool.json` |  15 类疾病兜底菜谱（4 餐/天） |
| `disease_icon/` | **新增**：15 个白色 svg 图标（导航 & 知识库使用） |

---

## 📄 pages/（7 页面，全部植入 `top_nav()`）
| 文件 | 作用 |
| ---- | ---- |
| `p00_intro.py` | 未登录落地页（不再改动） |
| `p00_auth.py` | 登录/注册（成功→p01_profile） |
| `p01_profile.py` | 三步档案：基础信息+15类病多选+过敏/菜系/睡眠/运动 |
| `01_overview.py` | 健康总览：15类SHAP雷达+子模型切换+建议卡片 |
| `02_nutrition.py` | 菜谱：AI 8 s 超时→兜底池+过敏/菜系/收藏+4餐横向滑 |
| `03_ai_doctor.py` | 左右气泡对话+15类快捷问题+流式输出+知识引用 |
| `p04_knowledge.py` | 双Tab：①疾病树（15类）②data_viz四图（训练集洞察） |
| `p05_me.py` | 个人中心：富文本笔记/CSV导出(zip)/注销 |

---

## 🧩 components/（6 个，零重复）
| 文件 | 作用 |
| ---- | ---- |
| `top_nav.py` | **新增**：全站顶部导航（5按钮，当前页玫瑰红高亮） |
| `shap_plot.py` | SHAP 瀑布 & 条形封装（01_overview 用） |
| `dish_camera.py` | 拍照识菜弹窗（02_nutrition 调用） |
| `ai_week_menu.py` | AI 生成函数（含 8 s 超时捕获） |
| `csv_export.py` | **新增**：统一多选 CSV → zip 导出（p05_me 调用） |
| `loader.py` | 加载动画 SVG（心脏脉冲，蓝→白） |

---

## 🔧 utils/（6 个，全站字典&兜底&可视化）
| 文件 | 作用 |
| ---- | ---- |
| `model.py` | 加载模型 + 预测（risk & subtype） |
| `crypto.py` | 哈希/加密（密码、敏感字段） |
| `viz.py` | 图表统一字体、调色盘、圆角样式 |
| `disease_dict.py` | **新增**：15 类疾病枚举（key/name/icon/diet_tip） |
| `recipe_fallback.py` | **新增**：读 recipe_pool.json → 过滤过敏/菜系/疾病 → 7 天菜单 |
| `data_viz.py` | **新增**：封装 cardio_train.csv 四图函数（供 p04 调用） |
utils/api.py（统一 API 初始化）
utils/cache.py (优化 API 加载速度)
utils/profile.py（统一加载 + 修复类型）
