根据您的项目结构，我来详细说明各个文件的作用：

## 🗂️ 项目文件结构说明

### **根目录文件**
- **`app.py`** - 主应用入口
  - 路由守卫：控制页面访问权限
  - 全局CSS样式注入
  - 应用初始化和配置

- **`train.py`** - 模型训练脚本
  - 训练XGBoost风险预测模型和亚型分类模型
  - 自动生成4张可视化图表到assets目录
  - 特征重要性分析和模型评估

- **`requirements.txt`** - 依赖包列表
  - 项目运行所需的所有Python包

- **`.env`** - 环境变量配置文件
  - 存储敏感信息（API密钥、数据库连接等）

### **📊 数据目录**
- **`data/cardio_train.csv`** - 训练数据集
  - 心脏健康相关的训练数据
  - 包含患者特征和心脏状况标签

### **🖼️ 资源目录 (assets/)**
- **`xgb_risk.pkl`** - 心脏风险预测模型
- **`xgb_subtype.pkl`** - 心脏亚型分类模型


### **📄 页面文件 (pages/)**
- **`p00_intro.py`** - 应用介绍页面
- **`00_auth.py`** - 用户认证页面（登录/注册）
- **`p01_profile.py`** - 用户个人资料管理
- **`01_overview.py`** - 健康数据总览仪表板
- **`02_nutrition.py`** - 营养与饮食建议
- **`03_ai_doctor.py`** - AI医生咨询和诊断
- **`p04_knowledge.py`** - 心脏健康知识库
- **`p05_me.py`** - 个人中心和设置

### **🧩 组件目录 (components/)**
- **`navbar.py`** - 导航栏组件
- **`floating_login.py`** - 浮动登录窗口
- **`carousel.py`** - 轮播图组件
- **`train_viz.py`** - 训练可视化组件
- **`shap_plot.py`** - SHAP分析图表组件
- **`dish_camera.py`** - 菜品拍照识别组件
- **`ai_week_menu.py`** - AI生成周菜单组件
- **`csv_importer.py`** - CSV数据导入组件
- **`export.py`** - 数据导出组件
- **`loader.py`** - 加载动画组件

### **🔧 工具函数目录 (utils/)**
- **`model.py`** - 模型推理和数据处理
  - 加载训练好的模型
  - 数据预处理和预测
- **`crypto.py`** - 加密解密工具
  - 密码哈希处理
  - 敏感数据加密
- **`viz.py`** - 可视化工具函数
  - 图表生成和样式设置
- **`prompt.py`** - AI提示词模板
  - 结构化AI对话提示
  - 健康建议生成模板

## 🔄 数据流概览
1. **用户输入** → 认证系统 → 个人资料
2. **健康数据** → 模型预测 → 风险分析
3. **AI分析** → 可视化展示 → 个性化建议
4. **持续监测** → 趋势分析 → 健康干预

这个架构支持一个完整的心脏健康监测和AI辅助诊断系统！
